## Docker + Apache + Laravel　開発環境
### image
- Apache: php:8.1-apache-bullseye
- Composer: 2.4.1
- Db: mysql:8.0.30
- Npm: 8.12.1

### install
- make build
- make up
- make composer
- make env
- make key
- make migration
